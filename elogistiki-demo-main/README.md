# e-logistiki — ΠΙΠΙΛΗ ΑΝΝΑ

Στατικός ιστότοπος για λογιστικό γραφείο. Αναπτύχθηκε με HTML, Tailwind CSS και Supabase.

## Σελίδες

| Αρχείο | Περιγραφή |
|---|---|
| `index.html` | Αρχική σελίδα |
| `services.html` | Υπηρεσίες (Προς Επιχειρήσεις & Προς Ιδιώτες) |
| `blog.html` | Blog / Νέα |
| `contact.html` | Επικοινωνία |
| `login.html` | Σύνδεση διαχειριστή |
| `admin-blog.html` | Διαχείριση blog |
| `post.html` | Σελίδα άρθρου |

## Τεχνολογίες

- [Tailwind CSS](https://tailwindcss.com/) (CDN)
- [Font Awesome 6.5](https://fontawesome.com/)
- [Supabase](https://supabase.com/) — backend για blog posts & authentication
